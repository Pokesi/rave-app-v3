# Rave Alpha App v3

## How to run locally

To fork the most up-to-date UI, use these steps:

1) Clone the repo & install deps
```
$ git clone https://github.com/rave-names/rave-app-v3 RaveSite3
$ # if you are using NPM
$ npm install
$ # if you are using Yarn
$ yarn
```

2) Run the server:
`npm run-script start`
